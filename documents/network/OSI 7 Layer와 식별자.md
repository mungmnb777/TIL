# OSI 7 Layer와 식별자

날짜: 2022년 12월 23일
카테고리: network

## MAC 주소 - Data Link 계층

NIC의 식별자

## IP 주소 - Network 계층

Host의 식별자(인터넷을 사용하는 컴퓨터)

## Port 번호 - Transport 계층

포트 번호는 크게 3가지로 나뉨

- 실제 Physical 레이어에서 작업하는 사람들 기준으로 포트라는 용어를 사용할 떄는 L2에서 사용하는 말임(네트워크 인터페이스)
- 네트워크를 관리하는 사람은 웹 서비스 식별자로 생각한다.
- 소프트웨어 개발하는 사람은 프로세스 식별자로 생각함.
